# Dotfiles

Nicolas' macOS development environment

## Structure

- `zsh/` - Shell configuration
- `ghostty/` - Terminal config
- `starship/` - Prompt theme
- `nvim/` - Neovim with LazyVim

## Install
```bash
cd ~/dotfiles
./install.sh
```

## Requirements
```bash
brew install neovim starship eza bat fzf zoxide ripgrep
```
